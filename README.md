# CartoGIS Portfolio 2026

This repository contains my final portfolio for the course **Cartographic visualization in GIS 2026**.

The portfolio explores GBIF mammal records in mainland Southeast Asia. The focus is on selected mammal groups that are relevant for forest environments and conservation: carnivores, primates, hoofed mammals, elephants, and pangolins.

## Published portfolio

The portfolio website is available here:

https://majahy.github.io/cartography-portfolio/

## Data and tools

The species occurrence records were downloaded from GBIF.org. The data was filtered and visualized with Python. The portfolio webpage was built with simple HTML/CSS and published with GitHub Pages.

## Note

GBIF records show where species have been documented. They do not represent the full or true distribution of the species.
